Splitter

Let's move slide bar!

Easy coding!

  1. Get area elements.
    var areaA = document.getElementById('area-a');
    var areaB = document.getElementById('area-b');
  2. And SPLIT!
    splitter = new Splitter(Splitter.Direction.Vertical, areaA, areaB);
  3. More SPLIT?
    Just
    areaC = document.getElementById('area-c');
    splitter.Split(Splitter.Side.A, Splitter.Direction.Horizontal, areaC);

What Horizontal Split?

+----------+     +----------+
| element  |     |   A      |
|          |  -> +----------+
|          |     |   B      |
+----------+     +----------+

What Vertical Split?

+----------+     +----+-----+
| element  |     |    |     |
|          |  -> | A  |  B  |
|          |     |    |     |
+----------+     +----+-----+

Can receive resize event!
